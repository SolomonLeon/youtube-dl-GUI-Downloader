Why I met Error "10060" or "10061"?

  • That's because you didn't connect to the Internet, or your service provider and government blocked the website you want to visit.
  • You can use some proxy software or change your hosts file.

How can I view my system proxy setting?

  • If you are using "PAC" mode, you can adjust your agent software to global mode, or follow the following steps.
  • Press "WIN"+"R", then enter "inetcpl.cpl" in the window that appears.
  • Go to the "connection" tab, and choise "LAN settings".
  • Copy the IP and port in the automatic configuration script setting like "127.0.0.1:1080", and paste them to the Downloader's proxy setting.
